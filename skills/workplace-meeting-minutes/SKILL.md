# 회의록 및 회의 준비 가이드

## 빠른 참조 (Quick Reference)

### 회의 유형별 특성
| 유형 | 목적 | 참석자 | 권장 시간 |
|------|------|--------|-----------|
| 킥오프 미팅 | 프로젝트 시작 | 전 팀원 | 60-90분 |
| 주간 팀 회의 | 진행 상황 공유 | 팀원 | 30-60분 |
| 의사결정 회의 | 결정 사항 도출 | 관련자 | 60분 |
| 브레인스토밍 | 아이디어 발산 | 관련자 | 60-90분 |
| 보고 회의 | 결과 공유 | 관련자+임원 | 30-60분 |
| 1:1 면담 | 개인 피드백 | 2명 | 30분 |

### 회의록 필수 항목
- 회의 일시/장소/참석자
- 회의 목적/주제
- 안건별 논의 내용
- 결정 사항 (명확한 주어+동사)
- Action Item (담당자 + 기한 필수)
- 다음 회의 일정

## 구현 가이드

### 효과적인 회의록 작성 4단계

**단계 1: 회의 중 기록**
- 중요 발언은 발언자 이름과 함께 기록
- 숫자/날짜는 반드시 정확히 기록
- 결정 사항은 회의 중 즉시 확인

**단계 2: 회의 직후 정리 (1시간 이내)**
- 기록된 내용 문장으로 정리
- Action Item 담당자/기한 재확인
- 불명확한 사항 즉시 확인

**단계 3: 배포 (24시간 이내)**
- 참석자 전원에게 이메일 발송
- 제목: [회의록] {회의명} (YYYY.MM.DD)
- 미참석자/관련자 추가 공유

**단계 4: 후속 관리**
- Action Item 진행 상황 추적
- 다음 회의에서 이전 회의록 Action Item 점검

### Action Item 작성 원칙
- **나쁜 예**: "마케팅팀에서 검토 예정"
- **좋은 예**: "마케팅팀 김철수 → 3월 제품 광고 예산안 작성 후 팀장 보고 (기한: 3월 10일)"
- 5W1H 원칙: 누가(Who), 무엇을(What), 언제까지(When) 필수

## 고급 패턴

### 온라인/화상 회의 추가 항목
```
[화상 회의 전용 섹션]
플랫폼: Zoom / Teams / Google Meet
회의 링크: {URL}
녹화 여부: O / X (저장 위치: {경로})
```

### 이의 없음 처리
- 회의록 배포 후 "X월 X일까지 이의 없으시면 확정으로 처리하겠습니다" 문구 추가
- 기한 내 이의 없으면 자동 확정 처리

### 영문 회의록 병기 (글로벌 팀)
- 주요 결정 사항은 국/영문 병기
- Action Item은 특히 영문 명확히 표기

### 회의 효율화 팁
- 시작 전 5분: 이전 Action Item 확인
- 종료 전 5분: 이번 회의 Action Item 정리
- 회의록은 회의 당일 또는 다음날 오전까지 배포
