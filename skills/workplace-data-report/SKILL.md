# 데이터 분석 리포트 작성 가이드

## 빠른 참조 (Quick Reference)

### 분석 리포트 유형
| 유형 | 목적 | 주요 독자 |
|------|------|-----------|
| 경영 대시보드 리포트 | 핵심 지표 현황 | 임원진 |
| 마케팅 성과 분석 | 캠페인/채널 효과 | 마케팅팀 |
| 매출/영업 분석 | 매출 동향, 예측 | 영업팀, 경영진 |
| 고객 분석 | 세그먼트, 이탈 | 마케팅, 전략팀 |
| 운영 효율 분석 | 비용, 생산성 | 운영팀, 경영진 |

### 인사이트 도출 프레임워크
1. **무슨 일이 있었나?** (What happened): 현황 기술
2. **왜 그랬나?** (Why): 원인 분석
3. **어떻게 할 것인가?** (So what): 시사점 및 권고

### 데이터 시각화 선택 기준
| 목적 | 차트 유형 |
|------|-----------|
| 시간 추이 | 꺾은선 그래프 |
| 구성 비율 | 파이/도넛 차트 |
| 비교 | 막대 그래프 |
| 분포 | 히스토그램, 박스플롯 |
| 상관관계 | 산점도 |
| 지역 분포 | 지도 시각화 |

## 분석 영역별 작성 패턴

### 매출 분석 패턴
```
■ 매출 현황 분석

1. 월별 매출 추이
   - X월 매출: {금액} (전월 대비 +X%, 전년 동기 대비 +X%)
   - 누적 매출: {금액} (목표 대비 X% 달성)
   - 추세: {상승/하락/보합} 추세 지속 중

2. 채널별 매출 구성
   - 온라인: {금액} (전체 X%)
   - 오프라인: {금액} (전체 X%)
   - 특이사항: {채널 변화 내용}

3. 제품별 성과
   TOP 3 제품:
   ① {제품명}: {금액} (점유율 X%)
   ② {제품명}: {금액} (점유율 X%)
   ③ {제품명}: {금액} (점유율 X%)

4. 분석 결론
   {핵심 인사이트 2-3줄}
```

```

   - 총 활성 고객: {명} (전월 대비 +X명)
   - 신규 고객: {명} | 이탈 고객: {명}
   - 고객 유지율(Retention): X%

   - 저활동 고객 (하위 50%): {명}

   - 재활성화 전략: {방안}
```

## 구현 가이드

### 데이터 분석 5단계 프로세스
1. **데이터 수집**: 출처 확인, 기간 정의
2. **데이터 정제**: 결측치/이상치 처리
3. **탐색적 분석**: 기술 통계, 시각화
4. **인사이트 도출**: 패턴 발견, 가설 검증
5. **보고서 작성**: 스토리라인 구성

### 숫자 표현 규칙
- 큰 수: 억/만 단위로 (3.5억, 1,250만)
- 비율: 소수점 1자리까지 (15.3%)
- 전월 대비는 반드시 화살표/색상으로 구분
- ▲ 상승 / ▼ 하락 / ▶ 보합

## 고급 패턴

### 통계적 유의성 표현
```
- 단순 수치 비교는 통계적 유의성 언급 주의
- 샘플이 작을 경우: "참고 수준의 분석"임을 명시
- 95% 신뢰구간 표기: X% (±Y%)
```

### 예측 분석 섹션
```
■ 다음 달 전망

낙관 시나리오 (+X%):
  조건: {조건}
  예상 매출: {금액}

기본 시나리오 (±X%):
  조건: 현재 추세 유지
  예상 매출: {금액}

비관 시나리오 (-X%):
  조건: {리스크 발생 시}
  예상 매출: {금액}
```
