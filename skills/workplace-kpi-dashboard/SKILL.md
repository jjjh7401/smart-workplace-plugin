# KPI 대시보드 설계 및 관리 가이드

## 빠른 참조 (Quick Reference)

### KPI vs OKR 비교
| 구분 | KPI | OKR |
|------|-----|-----|
| 초점 | 성과 측정 | 목표 달성 |
| 주기 | 월간/분기 | 분기/연간 |
| 특징 | 숫자 중심 | 도전적 목표 |
| 적합 | 운영 관리 | 성장 전략 |

### SMART KPI 원칙
- **Specific**: 구체적 (무엇을 측정하는가?)
- **Measurable**: 측정 가능 (어떻게 수치화?)
- **Achievable**: 달성 가능 (현실적인가?)
- **Relevant**: 관련성 (비즈니스와 연결되는가?)
- **Time-bound**: 기한 있음 (언제까지?)

## 업종별 핵심 KPI

### 전사 공통 KPI
```
재무 지표:
- 매출액 / 매출 성장률 (전월비/전년비)
- 영업이익률
- 비용 절감률
- 현금 흐름

고객 지표:
- 고객 수 (신규/유지/이탈)
- 고객 만족도 (NPS, CSAT)
- 고객 생애 가치 (LTV)
- 고객 획득 비용 (CAC)

운영 지표:
- 직원 생산성 (1인당 매출)
- 업무 처리 속도
- 오류율 / 불량률
```

### 이커머스/쇼핑몰 KPI
```
- GMV (총 거래액)
- 전환율 (Conversion Rate) 목표: 2-3%
- 평균 주문 금액 (AOV)
- 장바구니 이탈률 목표: 70% 이하
- 반품률
- 재구매율
- ROAS (광고 투자 수익률) 목표: 4이상
```

### SaaS/IT 서비스 KPI
```
- MAU/DAU (월간/일간 활성 사용자)
- 무료→유료 전환율 목표: 2-5%
- 구독 갱신율 목표: 85% 이상
- MRR (월간 반복 수익)
- Churn Rate (이탈률) 목표: 월 2% 이하
- NPS (순 추천 지수) 목표: 50 이상
- 티켓 해결 시간
```

### 제조업 KPI
```
- 생산량 / 생산 효율
- 불량률 목표: 1% 이하
- 가동률 목표: 85% 이상
- 납기 준수율 목표: 95% 이상
- 재고 회전율
- 원가율
```

## OKR 설계 방법

### OKR 템플릿
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
{팀명} 2024년 Q{분기} OKR
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Objective 1: {도전적이고 영감을 주는 목표}

  KR 1.1: {측정 가능한 결과 1}
          현재: {현재값} → 목표: {목표값} (X월 X일까지)

  KR 1.2: {측정 가능한 결과 2}
          현재: {현재값} → 목표: {목표값}

  KR 1.3: {측정 가능한 결과 3}
          현재: {현재값} → 목표: {목표값}

Objective 2: {두 번째 목표}

  KR 2.1: ...
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
달성 기준: KR 점수 0.7 이상 = 성공
           KR 점수 1.0 = 목표 낮게 설정
```

## 대시보드 설계

### 경영진 대시보드 레이아웃
```
┌─────────────────────────────────────────────┐
│  [로고]     {기간} 경영 현황 대시보드           │
├──────────┬──────────┬──────────┬────────────┤
│ 매출액   │ 영업이익  │ 신규고객  │ 목표달성률  │
│ XX억     │ X억(X%)  │ XXX명     │ XX%        │
│ ▲X%      │ ▲X%     │ ▼X%      │            │
├──────────┴──────────┴──────────┴────────────┤
│          월별 매출 추이 (꺾은선 그래프)          │
│                                              │
├─────────────────┬────────────────────────────┤
│  채널별 매출 구성 │   Top 5 제품 / 지역         │
│  (파이 차트)     │   (막대 그래프)              │
├─────────────────┴────────────────────────────┤
│  이슈/알림: ⚠️ {이슈 내용}                     │
└─────────────────────────────────────────────┘
```

### 대시보드 색상 코드 (신호등 시스템)
```
초록(달성): RGB(39, 174, 96) #27AE60
  - 목표 대비 100% 이상

노랑(주의): RGB(243, 156, 18) #F39C12
  - 목표 대비 80-99%

빨강(미달): RGB(231, 76, 60) #E74C3C
  - 목표 대비 80% 미만
```

## 구현 가이드

### KPI 설계 5단계
1. **비즈니스 목표 명확화**: 올해 회사가 달성해야 할 것?
2. **핵심 동력 파악**: 그 목표를 이루는 핵심 요인?
3. **측정 지표 선정**: 각 동력을 어떻게 측정?
4. **목표값 설정**: 과거 실적 + 성장률 기반
5. **측정 주기 결정**: 실시간/일/주/월

### KPI 보고서 주간 작성 예시
```
[주간 KPI 리포트]
기간: X월 X주 (X.XX ~ X.XX)

| KPI | 목표 | 실적 | 달성률 | 전주비 | 상태 |
|-----|------|------|--------|--------|------|
| 매출 | 1억 | 8,500만 | 85% | +5% | 🟡 |
| 신규고객 | 50명 | 58명 | 116% | +15% | 🟢 |
| 이탈률 | 3% | 4.2% | - | +0.3%p | 🔴 |

이번 주 핵심 인사이트:
- {주목할 지표 변화}
- {원인 분석}
- {다음 주 포커스}
```

## 고급 패턴

### 복합 지표 (Composite KPI)
```
고객 건강도 지수 (CHI):
= (구매 빈도 × 0.4) + (평균 구매액 × 0.3) + (NPS × 0.3)

해석:
- 80이상: 건강한 고객
- 60-79: 일반 관리
- 60미만: 이탈 위험 집중 관리
```

### Leading vs Lagging 지표
```
Lagging (후행 지표): 결과 측정
- 매출, 이익, 고객 수

Leading (선행 지표): 미래 예측
- 리드 수, 상담 건수, 시험 가입자
- → Leading을 통해 Lagging 예측 및 조기 대응
```
