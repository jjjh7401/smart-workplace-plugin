# SNS 및 블로그 콘텐츠 작성 가이드

## 빠른 참조 (Quick Reference)

### 플랫폼별 특성 비교
| 플랫폼 | 주요 사용자 | 콘텐츠 형식 | 최적 글자수 |
|--------|-----------|------------|------------|
| 인스타그램 | 20-40대 | 이미지+짧은 글 | 150-300자 |
| 네이버 블로그 | 전연령 | 글+이미지 | 1,000-3,000자 |
| 카카오스토리 | 30-50대 | 일상 공유 | 200-500자 |
| 링크드인 | 직장인 | 전문 인사이트 | 300-1,200자 |
| 유튜브 설명 | 전연령 | 영상 보조 | 200-500자 |

### 콘텐츠 유형별 목적
- **교육/정보**: 팔로워에게 가치 제공, 전문성 구축
- **스토리텔링**: 공감 유도, 브랜드 스토리
- **프로모션**: 제품/서비스 홍보, CTA 포함
- **인터랙션**: 질문, 투표, 댓글 유도
- **큐레이션**: 유용한 정보 모음, 리스트

## 플랫폼별 콘텐츠 템플릿

### 인스타그램 콘텐츠

#### 정보성 포스트
```
✅ {핵심 메시지 제목}

많은 분들이 모르시는 {주제} 꿀팁 {N}가지 👇

1️⃣ {팁 1}
→ {간단한 설명}

2️⃣ {팁 2}
→ {간단한 설명}

3️⃣ {팁 3}
→ {간단한 설명}

💡 이 내용이 도움됐다면
저장하고 필요할 때 꺼내보세요!

━━━━━━━━━━━━━━
#{관련 해시태그1} #{관련 해시태그2} #{관련 해시태그3}
#{업종 해시태그} #{브랜드 해시태그}
```

#### 스토리텔링 포스트
```
"{인상적인 첫 문장 또는 질문}"

{본문: 2-3단락}
- 배경/상황 설명
- 문제/갈등
- 해결/깨달음

{마무리 메시지 또는 CTA}

#스토리 #{관련 키워드}
```

### 네이버 블로그 포스트 구조

#### SEO 최적화 블로그 글
```
[제목] {키워드 포함 제목} - {차별화 포인트}
예: "엑셀 VLOOKUP 완벽 정리 - 초보도 5분 만에 이해하는 방법"

[도입부 - 200자 이내]
{독자가 겪는 문제 공감 + 이 글에서 얻을 것}

[목차]
1. {소제목 1}
2. {소제목 2}
3. {소제목 3}

[본문 섹션 구성]
## {소제목 1}
{설명 + 이미지/스크린샷}

## {소제목 2}
{설명 + 실용적 예시}

## {소제목 3}
{심화 내용 또는 꿀팁}

[마무리]
{핵심 요약 3줄}
{다음 포스팅 예고 또는 관련 글 링크}

[키워드 해시태그]
#네이버블로그 #{메인 키워드} #{관련 키워드}
```

### 링크드인 포스트

#### 전문 인사이트 포스트
```
{충격적 통계 또는 반전 있는 첫 문장}

저는 {X}년간 {업종/직무}에서 일하면서
이런 사실을 깨달았습니다.

{핵심 내용 3-5줄}
→ {구체적 사례나 데이터}
→ {시사점}

이것이 중요한 이유:
✅ {이유 1}
✅ {이유 2}
✅ {이유 3}

여러분의 경험은 어떤가요?
댓글로 공유해 주세요 👇

#직장생활 #{산업} #{전문분야}
```

## 해시태그 전략

### 인스타그램 해시태그 구성 (30개 제한)
```
대형 해시태그 (100만+): 3-5개
  #직장생활 #비즈니스 #마케팅 #취업

중형 해시태그 (10만-100만): 10-15개
  #업무효율 #직장인꿀팁 #엑셀 #PPT

소형 해시태그 (1만-10만): 5-10개
  #{세부 키워드} #{브랜드 관련}

브랜드 전용: 1-2개
  #{회사명} #{캠페인명}
```

### 네이버 블로그 키워드 전략
```
제목에 반드시 포함: 메인 키워드 1-2개
본문에 자연스럽게 포함: 3-5회
소제목에 키워드 변형형 포함

검색량 확인 방법:
- 네이버 키워드 도구 (searchad.naver.com)
- 목표: 월 1,000-10,000 검색, 발행 문서 1,000건 이하
```

## 구현 가이드

### 콘텐츠 캘린더 주간 계획 예시
```
월: 정보성 콘텐츠 (팁/가이드)
수: 스토리텔링 (사례/후기)
금: 프로모션 또는 인터랙션 (질문/이벤트)
일: 리마인더 또는 큐레이션

※ 플랫폼마다 최적 게시 시간 다름
인스타: 점심(12-13시), 저녁(20-22시)
링크드인: 화-목 아침(8-10시)
네이버 블로그: 오전(10-12시)
```

### 콘텐츠 작성 7단계
1. 타겟 독자 정의 (페르소나)
2. 그들의 핵심 고민/질문 파악
3. 플랫폼 선택
4. 콘텐츠 유형 결정
5. 핵심 메시지 1줄 정리
6. 초안 작성 (완벽보다 속도)
7. 편집 및 해시태그 추가

## 고급 패턴

### 바이럴 요소 체크리스트
- [ ] 첫 문장이 스크롤을 멈추게 하는가?
- [ ] 공유하고 싶은 유용한 정보가 있는가?
- [ ] 감정적 반응(공감, 놀라움, 웃음)을 유발하는가?
- [ ] 저장할 만한 "나중에 쓸 것 같은" 내용인가?
- [ ] CTA(댓글, 공유, 저장)가 있는가?

### A/B 테스트 방법
```
같은 내용을 두 가지 버전으로:
A: 통계/숫자로 시작
B: 질문으로 시작

2주 후 성과 비교:
- 도달률, 좋아요, 저장, 댓글 수 측정
```
