# 사업계획서 작성 가이드

## 빠른 참조 (Quick Reference)

### 사업계획서 유형별 구성
| 유형 | 분량 | 핵심 포인트 |
|------|------|-------------|
| VC/엔젤 투자유치 | 10-20페이지 | 시장 크기, 팀 역량, 성장성 |
| 정부지원사업 | 20-40페이지 | 사업화 가능성, 사회적 가치 |
| 은행/금융기관 | 10-15페이지 | 재무 안정성, 상환 능력 |
| 내부 전략 기획 | 제한 없음 | 실행 가능성, ROI |

### 투자자가 가장 보는 것
1. 팀 (Team): 이 팀이 해낼 수 있는가?
2. 시장 (Market): 충분히 큰 시장인가?
3. 제품 (Product): 차별화된 솔루션인가?
4. 견인력 (Traction): 이미 성과가 있는가?

## 사업계획서 표준 구조

### 목차
```
1. 사업 개요 (Executive Summary)
2. 문제 정의 및 시장 분석
3. 솔루션 및 제품/서비스
4. 비즈니스 모델
5. 시장 전략 (Go-to-Market)
6. 경쟁 분석
7. 팀 소개
8. 재무 계획
9. 투자 유치 계획 (해당 시)
10. 실행 로드맵
```

## 섹션별 작성 가이드

### 1. 사업 개요 (Executive Summary)
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
사업 개요

회사명: {회사명}
설립일: {날짜}
대표자: {이름}
업종: {업종}
주소: {주소}

한 줄 소개:
"{타겟 고객}이 겪는 {문제}를 {솔루션}으로 해결하여
{비즈니스 성과}를 창출하는 회사입니다."

핵심 지표:
- 목표 시장: {TAM 규모}
- 현재 단계: {아이디어/MVP/베타/런칭/성장}
- 누적 매출/사용자: {현황}
- 투자 유치 목표: {금액} (해당 시)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 2. 문제 정의 및 시장 분석
```
■ 문제 정의

현재 {타겟 고객}은 다음과 같은 문제를 겪고 있습니다:

문제 1: {구체적 문제}
  → 현재 대안: {기존 해결책의 한계}
  → 비용/불편: {문제의 크기 정량화}

문제 2: {구체적 문제}
  → ...

■ 시장 규모 분석

TAM (전체 시장): {규모} 억/조 원
  - 근거: {출처 및 산정 방법}

SAM (서비스 가능 시장): {규모}
  - 우리가 접근 가능한 세그먼트

SOM (실질 목표 시장): {규모}
  - 3년 내 목표 점유 시장

■ 시장 트렌드
- {트렌드 1}: {설명 및 수치}
- {트렌드 2}: {설명 및 수치}
```

### 3. 솔루션 및 제품
```
■ 핵심 솔루션

"{고객의 {문제}를 {방법}으로 해결하는 {제품명}}"

핵심 기능:
① {기능 1}: {고객 가치}
② {기능 2}: {고객 가치}
③ {기능 3}: {고객 가치}

차별화 포인트 (기존 대비):
| 기준 | 기존 방식 | 우리 솔루션 |
|------|----------|------------|
| {기준 1} | {기존} | {우리} |
| {기준 2} | {기존} | {우리} |

기술적 혁신 (해당 시):
- 특허/IP: {내용}
- 핵심 기술: {설명}
```

### 4. 비즈니스 모델
```
■ 수익 구조

주요 수익원:
① {수익원 1}: {단가} × {예상 고객수} = {연 매출}
② {수익원 2}: {구독/수수료/광고 등}

단위 경제학 (Unit Economics):
- CAC (고객 획득 비용): {금액}
- LTV (고객 생애 가치): {금액}
- LTV/CAC 비율: {비율} (3 이상 권장)
- 매출 총이익률: {%}
- BEP (손익분기점): {시점 또는 고객 수}
```

```

      | 1년차 | 2년차 | 3년차
------|-------|-------|-------

매출원가| X억  |  X억  |  X억
영업비용| X억  |  X억  |  X억
영업이익| X억  |  X억  |  X억
순이익  | X억  |  X억  |  X억

주요 가정:
- 고객 획득: 월 X명 (성장률 X%/월)
- 평균 거래액: X원
- 이탈률: X%/월
```

## 구현 가이드

### 사업계획서 작성 프로세스
1. **문제/솔루션 핏 검증**: 고객 인터뷰 20명 이상
2. **시장 조사**: 업계 리포트, 통계청 데이터 활용
3. **경쟁사 분석**: 직접/간접 경쟁사 포지셔닝 맵
4. **팀 구성 정리**: 각 멤버의 역할과 강점
5. **재무 모델링**: 보수/중간/낙관 시나리오
6. **스토리라인 구성**: 투자자의 관점에서 재구성

### 정부지원사업 신청 특화 팁
- 평가위원이 300개 신청서를 검토: 첫 페이지가 핵심
- 사업화 가능성 > 기술 혁신성 강조
- 사회적 가치/고용 창출 언급 필수
- 기존 유사 수상 경력 있으면 강조

## 고급 패턴

### 투자자용 핵심 나래이티브
```
투자자가 원하는 스토리:
1. "왜 지금인가?" - 시장 타이밍
2. "왜 이 팀인가?" - 팀 경쟁력
3. "왜 이 방식인가?" - 솔루션의 불가피성
4. "얼마나 커질 수 있나?" - 성장 가능성
```

### VC 투자 단계별 기대치
```
Pre-seed: 아이디어 + 팀 역량
Seed: MVP + 초기 고객 (PMF 탐색 중)
Series A: 성장 공식 검증 + 재현 가능한 성장
Series B+: 시장 지배력 + 수익성 경로
```
