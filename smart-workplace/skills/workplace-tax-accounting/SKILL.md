# 한국 세무/회계 기초 가이드

> **면책 고지**: 이 스킬은 세무 기초 교육용 정보를 제공합니다.
> 실제 세무 신고 및 중요 세무 사항은 반드시 세무사의 도움을 받으세요.

## 빠른 참조 (Quick Reference)

### 주요 세금 신고 일정
| 세금 종류 | 신고 시기 | 납부 기한 |
|-----------|-----------|-----------|
| 부가가치세 (1기) | 7월 1-25일 | 7월 25일 |
| 부가가치세 (2기) | 다음해 1월 1-25일 | 1월 25일 |
| 종합소득세 | 5월 1-31일 | 5월 31일 |
| 법인세 | 사업연도 종료 후 3개월 | 3월 31일 (12월 결산) |
| 원천세 | 다음 달 10일까지 | 매월 |
| 지방세 | 세목별 상이 | - |

### 세금계산서 발행 원칙
- 공급일이 속하는 달의 다음 달 10일 이내 발행
- 전자세금계산서: 직전연도 공급가액 3억 이상 의무
- 수정세금계산서: 환입, 내국신용장 개설 등 사유 발생 시

## 주요 세금 개념

### 부가가치세 (VAT)

#### 기본 개념
```
부가가치세 = 매출세액 - 매입세액

예시:
- 판매 시: 공급가액 100만 원 + 부가세 10만 원 = 110만 원 청구
- 매출세액: 10만 원 (납부 의무)
- 매입세액: 구매 시 지불한 부가세 (공제 가능)
- 납부세액: 매출세액 - 매입세액

세율:
- 일반: 10%
- 영세율: 0% (수출, 국제 운송 등)
- 면세: 부가세 없음 (생활 필수품, 의료, 교육 등)
```

#### 세금계산서 구조
```
전자세금계산서 필수 기재 항목:
① 공급자 등록번호, 상호, 성명, 주소
② 공급받는 자 등록번호, 상호, 성명
③ 작성 연월일
④ 공급가액 (부가세 제외 금액)
⑤ 세액 (공급가액의 10%)

품목별 기재:
- 품목명, 규격, 수량, 단가, 공급가액, 세액
```

### 종합소득세 (개인 사업자)

#### 소득 종류와 합산
```
종합과세 대상 소득 6가지:
① 이자소득
② 배당소득
③ 사업소득 (개인 사업자 주요 항목)
④ 근로소득
⑤ 연금소득
⑥ 기타소득

※ 분리과세 소득은 종합소득에 합산 안 됨
```

#### 세율 (2024년 기준)
```
과세표준          | 세율 | 누진공제
------------------|------|--------
1,400만 원 이하   |  6%  |    -
1,400-5,000만     | 15%  |  126만
5,000만-8,800만   | 24%  |  576만
8,800만-1.5억     | 35%  | 1,544만
1.5억-3억         | 38%  | 1,994만
3억-5억           | 40%  | 2,594만
5억-10억          | 42%  | 3,594만
10억 초과         | 45%  | 6,594만
```

#### 주요 공제 항목
```
기본공제: 본인 150만 원, 배우자/부양가족 각 150만 원
추가공제: 경로우대 100만 원, 장애인 200만 원
특별공제: 국민연금, 건강보험료, 고용보험료
의료비공제: 총급여의 3% 초과분
교육비공제: 대학교 900만 원, 중고 300만 원
주택 관련: 주택임차 차입금 이자 등
```

### 법인세

#### 세율 (2024년)
```
과세표준              | 세율
---------------------|------
2억 원 이하           |  9%
2억-200억             | 19%
200억-3,000억         | 21%
3,000억 초과          | 24%
```

#### 법인세 신고 흐름
```
결산 → 세무조정 → 과세표준 계산 → 세율 적용 → 신고/납부

결산: 기업회계기준에 따른 당기순이익 계산
세무조정: 세법 기준으로 차이 항목 조정
  - 익금산입: 회계상 수익이지만 세법상 추가 수익
  - 손금산입: 세법상 비용으로 인정 (회계는 비용 아님)
  - 익금불산입/손금불산입도 있음
```

## 세무 서류 작성 가이드

### 영수증/증빙 관리
```
공제 가능 증빙 (우선순위):
1. 세금계산서 (가장 확실)
2. 현금영수증 (사업자용)
3. 신용카드 매출전표 (사업용 카드)
4. 간이영수증 (3만 원 미만만 인정)

주의:
- 사업 관련 지출만 공제 가능
- 접대비: 연간 3,600만 원 한도
- 증빙 보관: 5년간 의무
```

### 원천세 신고 흐름
```
원천징수 의무:
- 직원 급여 지급 시: 소득세 + 지방소득세 원천징수
- 프리랜서 지급 시: 3.3% 원천징수 (소득세 3% + 지방세 0.3%)
- 이자, 배당 지급 시: 각 세율로 원천징수

신고 방법:
홈택스 접속 → 신고/납부 → 원천세 →
근로/사업소득 원천세 신고 → 지급내역 입력 → 제출
```

## 구현 가이드

### 세무 캘린더 연간 관리
```
1월: 부가세 2기 신고, 연말정산 간소화 서비스
2월: 연말정산 완료, 근로소득 지급명세서 제출
3월: 법인세 신고 (12월 결산 법인)
4월: 개인사업자 부가세 예정 신고 (1기 예정)
5월: 종합소득세 신고 (개인 사업자, 근로외소득)
7월: 부가세 1기 신고
10월: 법인 부가세 예정 신고
```

### 홈택스 활용 팁
```
주요 메뉴:
- 전자세금계산서 발행: 세금계산서/계산서 → 발행
- 현금영수증 조회: 현금영수증 → 사용내역
- 세금 신고: 신고/납부 → 해당 세목
- 증명서 발급: 민원증명 → 각종 증명서

앱: 손택스 (모바일)
```

## 고급 패턴

### 절세 합법적 방법
```
개인 사업자:
- 노란우산공제 (연 최대 500만 원 소득공제)
- 중소기업 특별 세액감면 활용
- 청년/창업 감면 적용 여부 확인

법인:
- 가족 임원 급여 설정 (적정 범위 내)
- R&D 세액공제 (25-40%)
- 설비투자 세액공제 활용
```

### 세무조사 대비
```
대비 사항:
- 증빙 5년 보관 (전자 또는 종이)
- 법인카드/사업용 계좌 분리 사용
- 가지급금 최소화
- 개인 경비와 법인 경비 철저 분리
```
