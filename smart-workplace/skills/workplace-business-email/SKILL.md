# 비즈니스 이메일 작성 가이드

## 빠른 참조 (Quick Reference)

### 이메일 유형
- **요청/승인**: 협조 요청, 승인 요청, 자료 요청
- **보고**: 업무 결과, 현황 보고, 진행 상황 공유
- **안내**: 일정 안내, 변경 사항 공지, 행사 안내
- **회신**: 문의 답변, 확인 회신
- **감사**: 협조 감사, 미팅 후 감사
- **사과**: 실수 사과, 지연 사과
- **거절**: 정중한 거절, 일정 조율 불가

### 격식 수준
| 대상 | 격식 수준 | 인사말 예시 |
|------|-----------|------------|
| 외부 거래처 | 최고 격식 | 안녕하십니까 |
| 상급자/임원 | 격식체 | 안녕하세요 |
| 동료/팀원 | 일반체 | 안녕하세요 / 수고 많으십니다 |

### 기본 구조
```
수신: 이름/직책
제목: [유형] 핵심 내용 (명확하고 간결하게)
---
인사말 → 자기 소개(외부) → 이메일 목적 → 본문 → 요청/기대 사항 → 맺음말 → 서명
```

## 이메일 유형별 템플릿

### 1. 미팅 요청 이메일
```
제목: [미팅 요청] {주제} 관련 미팅 요청드립니다

안녕하세요, {회사명} {이름}입니다.

{관계/배경 한 줄 설명}

다름이 아니라, {주제}와 관련하여 미팅을 요청드리고자 연락드립니다.

- 목적: {미팅 목적}
- 예상 소요 시간: {시간}
- 희망 일정: {날짜 1} 또는 {날짜 2} (편하신 시간에 맞춰드리겠습니다)
- 장소: {장소 또는 화상 미팅}

바쁘신 중에 요청드려 죄송합니다.
일정 확인 후 회신 주시면 감사하겠습니다.

감사합니다.
{이름} 드림
```

### 2. 자료 요청 이메일
```
제목: [자료 요청] {자료명} 공유 요청드립니다

안녕하세요, {이름}입니다.

{배경 설명 1-2문장}

{자료명}을 공유해 주시면 감사하겠습니다.

- 필요 자료: {구체적 자료명}
- 필요 이유: {사용 목적}
- 희망 기한: {날짜}까지

무리한 부탁인 경우 말씀해 주시면 이해하겠습니다.
바쁘신 중에 요청드려 감사합니다.

{이름} 드림
```

### 3. 결과 보고 이메일
```
제목: [보고] {프로젝트/업무명} 결과 보고

안녕하세요, {이름}입니다.

{업무/프로젝트명} 관련 결과를 보고드립니다.

■ 완료 내용
- {완료 항목 1}
- {완료 항목 2}

■ 주요 성과
- {성과 지표 또는 결과}

■ 후속 조치 (필요 시)
- {다음 단계}

검토 후 피드백 주시면 감사하겠습니다.

{이름} 드림
```

### 4. 감사 이메일
```
제목: [감사] {미팅/협조} 감사드립니다

안녕하세요, {이름}입니다.

지난 {날짜} {상황}에 {감사 내용}해 주셔서 진심으로 감사드립니다.

{구체적 감사 내용 1-2문장}

앞으로도 좋은 관계 유지하며 상호 발전할 수 있기를 기대합니다.
다시 한번 감사드립니다.

{이름} 드림
```

### 5. 정중한 거절 이메일
```
제목: Re: {원래 제목}

안녕하세요, {이름}입니다.

{제안/요청}에 대해 검토해 주셔서 감사합니다.

아쉽게도 {이유: 내부 일정 문제 / 예산 상황 / 역량 범위 초과} 등으로 인해
이번에는 함께하기 어려울 것 같습니다.

더 좋은 기회에 함께할 수 있기를 바랍니다.
이해해 주셔서 감사합니다.

{이름} 드림
```

### 6. 수신 확인 회신 이메일
```
제목: Re: {원래 제목}

안녕하세요, {이름}입니다.

{내용} 잘 수신하였습니다. 감사합니다.

검토 후 {날짜}까지 / 빠른 시일 내에 회신드리겠습니다.

{이름} 드림
```

## 경어 및 핵심 표현 가이드

### 인사말
- **처음 연락 시**: "처음 연락드립니다. 저는 {회사} {이름}입니다."
- **일반 인사**: "안녕하세요, {이름}입니다."
- **재연락 시**: "항상 감사드립니다."
- **상급자에게**: "수고 많으십니다."

### 요청 표현 (격식 순)
- ~해 주시면 감사하겠습니다
- ~부탁드립니다
- ~검토 부탁드립니다
- ~확인 부탁드립니다

### 마무리 표현
- "바쁘신 중에 감사드립니다."
- "검토 후 회신 주시면 감사하겠습니다."
- "좋은 하루 되세요."
- "항상 건강하세요."

### 첨부파일 언급
- "첨부 파일을 참조해 주시기 바랍니다."
- "관련 자료를 첨부하였으니 확인 부탁드립니다."
- "파일 2개를 첨부하였습니다: {파일명1}, {파일명2}"

## 구현 가이드

### 이메일 작성 단계
1. **목적 파악**: 이 이메일로 얻고자 하는 것은?
2. **대상 파악**: 수신자와의 관계, 격식 수준 결정
3. **핵심 메시지 1줄 정리**: 한 문장으로 요약 가능해야 함
4. **제목 작성**: [유형] + 핵심 내용 (20자 이내 권장)
5. **본문 작성**: 두괄식 (결론 먼저), 간결하게
6. **검토**: 맞춤법, 경어, 첨부파일 확인

### 제목 작성 원칙
- [요청], [보고], [안내], [회신], [감사], [긴급] 등 태그 활용
- 30자 이내 권장
- 구체적 내용 포함 (예: "보고" X → "[보고] 3월 매출 현황 보고" O)

### CC/BCC 활용
- **CC(참조)**: 내용을 알아야 하는 관련자 (직속 상사, 관련 팀원)
- **BCC(숨은 참조)**: 발신 사실을 수신자에게 알리지 않을 때

## 고급 패턴

### 부탁/요청이 많을 때 구조화
```
요청 사항이 여러 가지인 경우 번호를 사용합니다:

요청드릴 사항이 3가지 있습니다:
① {요청 1} → {기한}
② {요청 2} → {기한}
③ {요청 3} → {기한}
```

### 긴급 이메일 처리
- 제목에 [긴급] 또는 [급] 태그 추가
- 첫 문장에 긴급성과 기한 명시
- 전화/메신저 후 이메일 발송 병행

### 외국어 혼용 규칙
- 국문 이메일에서 영어 단어: IT 용어, 고유명사는 영어 사용 가능
- 약어 처음 사용 시 괄호로 설명: CRM(고객관계관리시스템)

### 실수 대응 이메일
```
제목: [정정] 이전 이메일 내용 정정 안내

안녕하세요, {이름}입니다.

방금 발송한 이메일에서 오류가 있어 정정 안내드립니다.

- 잘못된 내용: {내용}
- 정정 내용: {내용}

혼란을 드려 죄송합니다.
```
