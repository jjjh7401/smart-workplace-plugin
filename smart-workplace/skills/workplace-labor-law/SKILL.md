# 근로기준법 가이드

> **면책 고지**: 이 스킬은 일반적인 근로기준법 안내 목적으로 제공됩니다.
> 개별 사안에 대한 법적 판단은 노무사 또는 변호사의 조언을 받으세요.

## 빠른 참조 (Quick Reference)

### 근로자 핵심 권리
| 권리 | 내용 | 위반 시 |
|------|------|---------|
| 최저임금 | 2024년: 시간당 9,860원 | 형사처벌 가능 |
| 주휴수당 | 주 15시간 이상 근무 시 1일치 추가 지급 | 임금체불 |
| 연차휴가 | 1년 미만: 월 1일 / 1년 이상: 15-25일 | 연차수당 지급 |
| 퇴직금 | 1년 이상, 주 15시간 이상 근무 시 | 지연이자 포함 지급 |
| 해고예고 | 30일 전 예고 또는 30일치 급여 | 해고예고수당 |
| 산재보험 | 업무 중 부상/질병 보상 | 사용자 책임 |

### 근무 시간 기준 (1주)
- 소정근로: 40시간 이하
- 연장근로: 최대 12시간 (총 52시간)
- 연장/야간/휴일 근무: 통상임금의 50% 가산

## 주요 근로 권리 상세

### 연차유급휴가

#### 발생 기준
```
1년 미만 근무 (입사 후):
- 매월 개근 시 1일씩 발생
- 최대 11일 (1년 미만)

1년 이상 근무:
- 1년 개근 시: 15일
- 3년 이상: 2년마다 1일 추가 (최대 25일)

사용 기한:
- 발생일로부터 1년 이내 사용
- 미사용 시: 연차수당으로 지급 (통상임금 × 미사용 일수)
```

#### 연차 거부 가능 여부
```
원칙: 근로자가 청구한 시기에 부여 의무
예외: 사업 운영에 막대한 지장이 있는 경우 시기 변경 가능
     (단, 연차 사용 자체 거부는 불법)

실무 팁:
- 연차 사용 계획서 제출 (분쟁 예방)
- 시기 변경 요청 시 사용자는 대체 일자 제시 의무
- 연차 사용 촉진제: 사용자가 촉구해도 미사용 시 수당 면제 가능
```

### 퇴직금

#### 계산 방법
```
퇴직금 = 1일 평균임금 × 30일 × (재직일수 ÷ 365)

1일 평균임금 = 퇴직 전 3개월간 임금 총액 ÷ 3개월간 총 일수

예시:
- 월 급여 300만 원 × 3개월 = 900만 원
- 3개월간 일수: 91일
- 1일 평균임금: 900만 / 91일 = 약 9.89만 원
- 근속 2년(730일) 퇴직금 = 9.89만 × 30 × (730÷365) = 약 593만 원
```

#### 퇴직금 지급 기한
```
퇴직일로부터 14일 이내 지급 (합의 시 기간 연장 가능)
지연 시: 연 20% 지연이자 발생 (소액사건심판법)
미지급 시: 고용노동부 진정 또는 민사소송
```

### 해고 보호

#### 정당한 해고 요건
```
해고의 정당한 이유:
① 근로자의 귀책사유 (중대한 취업규칙 위반, 횡령 등)
② 경영상 이유 (긴박한 경영상 필요 + 회피 노력 + 협의)

해고 절차:
① 해고 사유 서면 통지 (사유와 시기 명시)
② 30일 전 예고 또는 예고수당 지급
③ 취업규칙/단체협약상 절차 준수

부당해고 시 구제:
① 노동위원회 부당해고 구제신청 (해고일로부터 3개월 이내)
② 복직 또는 해고 기간 임금 지급 명령
③ 금전보상 명령 (복직 원하지 않는 경우)
```

#### 해고 금지 기간
```
절대 해고 금지:
- 업무상 부상/질병 치료 중 + 치료 후 30일
- 산전후 휴가 기간 + 휴가 후 30일
- 육아휴직 기간

주의: 해고 금지 기간 중 해고는 무효
```

## 직장 내 법률 상황별 대응

### 임금체불 발생 시
```
Step 1: 사용자에게 서면으로 지급 요청
Step 2: 지급 거부 시 → 고용노동부 진정 접수
  - 온라인: 고용노동부 홈페이지
  - 오프라인: 관할 지방노동청
Step 3: 민사소송 (소액사건심판 - 3,000만 원 이하 간이 절차)

소멸시효: 임금은 3년, 퇴직금은 3년
```

### 직장 내 괴롭힘
```
2019년 시행, 근로기준법 제76조의2

괴롭힘 정의:
직장에서의 지위 또는 관계 등의 우위를 이용하여
업무상 적정 범위를 넘어 신체적·정신적 고통 주는 행위

처리 절차:
① 회사 내부 신고 (취업규칙상 신고 절차)
② 고용노동부 신고
③ 사용자의 조치 의무: 조사 → 피해자 보호 → 가해자 징계

※ 신고를 이유로 한 불이익 취급 금지 (위반 시 처벌)
```

### 산업재해 (산재)
```
업무상 재해 인정 기준:
- 업무 중 사고
- 출퇴근 재해 (통상적인 경로)
- 업무상 질병 (직업병)

처리 절차:
① 사고 발생 즉시 회사에 보고
② 요양급여 신청 (근로복지공단)
③ 치료비 + 휴업급여 (평균임금의 70%) 지급

주의: 산재 신청은 근로자 권리 (회사 동의 불필요)
```

## 구현 가이드

### 주요 노동 관련 기관
```
고용노동부: https://www.moel.go.kr
  - 임금체불, 부당해고 진정
  - 전화: 1350 (고용노동부 상담 센터)

근로복지공단: https://www.kcomwel.or.kr
  - 산재보험, 퇴직연금

노동위원회: https://www.nlrc.go.kr
  - 부당해고 구제신청

대한법률구조공단: https://www.klac.or.kr
  - 무료 법률 상담 및 소송 지원
```

### 근로계약서 없을 때 대처
```
미작성 시 처벌: 사용자에게 500만 원 이하 과태료

증거 수집 방법:
- 근무 사실 증명: 급여 입금 내역, 카카오톡 대화, 업무 이메일
- 근무 시간 증명: 출퇴근 카드, 업무 지시 문자
- 임금 수준 증명: 동종 업계 임금, 채용공고
```

## 고급 패턴

### 프리랜서 vs 근로자 구분
```
근로자성 인정 기준 (대법원):
- 업무 내용을 사용자가 지정
- 취업규칙/복무규정 적용
- 지휘·감독 받음
- 근무 시간/장소 고정
- 보수의 성격 (급여 vs 사업소득)

프리랜서로 계약했더라도 근로자성 인정 가능
→ 퇴직금, 4대보험 소급 청구 가능
```

### 최저임금 계산
```
2024년 최저임금: 9,860원/시간

월 최저임금 (주 40시간 기준):
9,860원 × 209시간 = 2,060,740원

209시간 = (주 40시간 + 주휴 8시간) × 52주 ÷ 12개월
```
