# 한국 HR 문서 작성 가이드

## 빠른 참조 (Quick Reference)

### 주요 HR 문서 목록
| 문서 | 발급 시점 | 필수 여부 |
|------|-----------|-----------|
| 근로계약서 | 입사 시 | 법적 의무 |
| 재직증명서 | 요청 시 | 요청 즉시 발급 의무 |
| 경력증명서 | 퇴직 후 요청 시 | 요청 시 발급 의무 |
| 퇴직증명서 | 퇴직 시 | 요청 시 발급 의무 |
| 인사발령 통보서 | 발령 시 | 권장 |
| 연봉 계약서 | 연간 | 권장 |
| 업무 위임장 | 필요 시 | 선택 |

### 근로계약서 법정 필수 기재 항목 (근로기준법 제17조)
1. 임금 (구성 항목, 계산 방법, 지급 방법)
2. 소정근로시간
3. 휴일
4. 연차 유급휴가
5. 취업 장소
6. 업무 내용
7. 취업규칙 관련 사항

## HR 문서 템플릿

### 1. 표준 근로계약서
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
근    로    계    약    서
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

(이하 "사용자"라 한다)와
성명 {이름} (이하 "근로자"라 한다)은
아래와 같이 근로계약을 체결한다.

제1조 (계약기간)
근로계약 기간: {시작일}부터 {종료일}까지
※ 기간의 정함이 없는 경우: "기간의 정함이 없는 근로계약"

제2조 (근무 장소 및 업무)
- 근무 장소: {주소 및 부서}
- 담당 업무: {업무 내용}

제3조 (소정근로시간)
- 근무 시간: {시작 시간}부터 {종료 시간}까지
- 소정근로시간: 주 {시간}시간 (1일 {시간}시간)
- 휴게 시간: {시간} (점심 등)
- 근무일: 월~금 (주5일 근무)

제4조 (휴일 및 휴가)
- 주휴일: 매주 토요일, 일요일
- 법정 공휴일: 유급 처리
- 연차유급휴가: 근로기준법에 따름
  (1년 미만: 월 1일, 1년 이상: 15일~25일)

제5조 (임금)
- 기본급: 월 {금액}원
- 제수당: {수당 항목 및 금액} (해당 시)
- 연장/야간/휴일 수당: 근로기준법 기준 적용
- 지급일: 매월 {일}일
- 지급 방법: 근로자 명의 계좌 이체

제6조 (4대보험)
사용자는 근로자를 국민연금, 건강보험, 고용보험,
산재보험에 가입시킨다.

제7조 (퇴직금)
1년 이상 근무 시 퇴직금 지급 (근로기준법에 따름)
또는 퇴직연금 가입 여부: {DB형/DC형/해당 없음}

제8조 (비밀유지)
근로자는 업무상 알게 된 회사 정보를 외부에
누설하지 아니한다.

제9조 (계약 해지)
근로기준법에 따라 계약을 해지할 수 있다.

기타 이 계약서에 명시되지 않은 사항은
근로기준법 및 회사 취업규칙에 따른다.

{날짜}

사 용 자: {회사명} (대표이사: {이름}) (인)
          {주소/사업자번호}

근 로 자: {이름} (서명)
          생년월일: {날짜}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 구현 가이드

### 근로계약서 작성 시 자주 하는 실수
```
❌ 잘못된 예:
- 임금 총액만 기재 (각 구성 항목 미기재)
- 근무 시간 없이 "주 40시간 이내"만 기재
- 구두 계약 후 계약서 미작성

✅ 올바른 예:
- 기본급 XXX만 원 + 식대 10만 원 + 교통비 10만 원 = 총 XXX만 원
- 09:00 ~ 18:00 (휴게 12:00~13:00, 소정근로 8시간)
- 입사 당일 또는 입사 전에 서명 후 양측 각 1부 보관
```

### 4대보험 가입 의무
```
모든 근로자 (1주 소정근로 15시간 이상):
- 국민연금: 4.5% (근로자), 4.5% (사용자)
- 건강보험: 3.545% (근로자), 3.545% (사용자)
- 고용보험: 0.9% (근로자), 1.15%+ (사용자)
- 산재보험: 전액 사용자 부담 (업종별 상이)
```

## 고급 패턴

### 수습기간 조항 활용
```
수습 기간: 입사일로부터 3개월
수습 중 임금: 통상임금의 90% (최저임금 이상)

※ 수습 만료 후 별도 통보 없으면 정규직 전환으로 간주
※ 수습 종료 전 계약 해지: 30일 전 예고 또는 30일치 급여 지급
```

### 포괄임금제 주의사항
```
포괄임금제 적용 가능 조건:
- 근로시간 산정이 어려운 업무
- 감시/단속 직종

포괄임금제 금지 추세:
- 2021년 이후 법원: 사무직/생산직 포괄임금 무효 판결 증가
- 연장근로 시간 별도 기재 권장
```
