# CRM(고객관계관리) 전략 및 구현 가이드

## 빠른 참조 (Quick Reference)

### CRM 도구 선택 가이드
| 도구 | 특징 | 적합 대상 | 가격 |
|------|------|-----------|------|
| HubSpot (무료) | 올인원, 영문 중심 | 스타트업 | 무료~유료 |
| Salesforce | 강력한 커스터마이징 | 대기업 | 고가 |
| Zoho CRM | 가성비 | 중소기업 | 중간 |
| 구글 시트 | 간단, 무료 | 초기 스타트업 | 무료 |
| Notion | 유연한 구조 | 소규모 팀 | 무료~ |

### 한국 B2B 영업 특성
- 관계(Relationship) 중심: 담당자 개인 신뢰가 중요
- 의사결정 계층: 실무자→팀장→임원 다단계
- 긴 세일즈 사이클: 평균 3-12개월
- 명함 교환/접대 문화: 관계 유지 비용 중요
- 계절성: 연말/분기말 예산 집행 집중

## 구현 가이드

### 구글 시트로 간단 CRM 구축

#### 필수 시트 구성
```
시트 1: 고객사 목록
  컬럼: 회사명, 업종, 규모, 담당자, 연락처,
        파이프라인 단계, 예상 금액, 예상 계약일, 메모

시트 2: 활동 로그
  컬럼: 날짜, 고객사, 활동 유형, 내용, 결과, 다음 액션

시트 3: 파이프라인 현황
  피벗 테이블로 단계별 금액 합계 자동 계산

시트 4: 캘린더
  이번 주/다음 주 팔로업 필요 항목 자동 필터
```

### 고객 세그먼테이션 (ABC 분류)
```
A등급 (핵심 고객: 상위 20%, 매출 80%):
→ 월 1회 이상 직접 방문/미팅
→ 전담 담당자 배정

B등급 (잠재 성장 고객):
→ 분기 1회 미팅
→ 월 1회 이메일/전화

C등급 (소규모/잠재 고객):
→ 뉴스레터/정기 안내 자동화
→ 반기 1회 연락
```

## 고급 패턴

### 고객 이탈 신호 감지
```
주의 신호 (Early Warning):
- 담당자 연락 빈도 갑자기 감소
- 계약 갱신 6개월 전부터 무반응
- 불만 제기 증가
- 사용량/주문량 감소 트렌드

대응 방법:
1. 담당자 임원에게 직접 연락
2. 서비스 만족도 조사 제안
3. 특별 혜택/조건 검토
4. 이탈 사유 분석 및 개선
```

### MEDDIC 판매 방법론 적용
```
Metrics: 고객이 얻을 수치적 효과는?
Economic Buyer: 최종 의사결정자가 누구인가?
Decision Criteria: 평가 기준은 무엇인가?
Decision Process: 어떤 프로세스로 결정하는가?
Identify Pain: 해결하고자 하는 핵심 문제는?
Champion: 내부에서 우리를 지지해줄 사람은?
```
