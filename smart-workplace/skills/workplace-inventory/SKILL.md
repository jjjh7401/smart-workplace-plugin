# 재고관리 시스템 설계 가이드

## 빠른 참조 (Quick Reference)

### 재고관리 핵심 지표
| 지표 | 공식 | 목표 수준 |
|------|------|-----------|
| 재고 회전율 | 매출원가 ÷ 평균 재고 | 업종별 상이 |
| 재고 일수 | 365 ÷ 재고 회전율 | 최소화 |
| 안전 재고 | 리드타임 × 일평균 소비 × 안전계수 | 적정 수준 |
| 재고 정확도 | (실사 수량 ÷ 장부 수량) × 100 | 95% 이상 |
| 결품률 | 결품 건수 ÷ 총 주문 건수 × 100 | 1% 이하 |

### 재고관리 방법론
- **FIFO (선입선출)**: 먼저 입고된 것 먼저 출고 - 식품, 의약품
- **LIFO (후입선출)**: 나중 입고된 것 먼저 출고 - 원자재 (세무 목적)
- **FEFO (유통기한 우선)**: 유통기한 가까운 것 먼저 - 식품, 화장품
- **ABC 분류**: 매출 기여도에 따라 A/B/C 등급 관리

## 재고 관리 시스템 설계

### ABC 분류 방법
```
분류 기준: 매출액 기여도

A등급 (상위 품목: 매출 70%):
  - 전체 SKU의 약 10-20%
  - 일별 재고 모니터링
  - 안전 재고: 평균 소비량의 30-50%
  - 발주점: 높게 설정

B등급 (중간: 매출 20%):
  - 전체 SKU의 약 30-40%
  - 주별 모니터링
  - 안전 재고: 평균 소비량의 20-30%

C등급 (나머지: 매출 10%):
  - 전체 SKU의 약 50-60%
  - 월별 모니터링
  - 안전 재고: 최소 또는 수주 생산 고려
```

### 발주점(Reorder Point) 계산
```
발주점 = (일평균 소비량 × 리드타임) + 안전재고

예시:
- 일평균 소비: 50개
- 리드타임: 7일
- 안전계수: 1.65 (서비스 수준 95%)
- 리드타임 수요 편차: 30개

안전재고 = 1.65 × 30 = 50개
발주점 = (50 × 7) + 50 = 400개

→ 재고가 400개 이하로 떨어지면 발주
```

## 재고 실사 프로세스

### 정기 실사 절차 (월간/분기별)
```
사전 준비 (D-3):
□ 실사 날짜/시간 공지
□ 실사 대상 품목 목록 출력
□ 실사 팀 구성 (최소 2인 1조 권장)
□ 창고 정리 정돈

실사 당일:
□ 시스템 입출고 마감 (실사 시작 전)
□ 조별 구역 배정
□ 실물 수량 카운트 (바코드/수기)
□ 1차 카운트 완료 후 차이 항목 재카운트
□ 차이 보고서 작성

실사 후 (D+1):
□ 차이 원인 분석
□ 재고 조정 처리 (시스템 반영)
□ 차이 원인에 따른 개선 조치
□ 실사 결과 보고서 제출
```

## 구현 가이드

### 엑셀 기반 재고관리 시스템 구축

#### 필수 시트 구성
```
Sheet 1: 품목 마스터
  - 품목코드, 품목명, 단위, 분류, 안전재고, 발주점, 표준단가

Sheet 2: 현재고
  - 품목코드 기준 현재 재고 (자동 계산)
  =SUMIF(입고!D:D, A2, 입고!E:E) - SUMIF(출고!D:D, A2, 출고!E:E)

Sheet 3: 입고 대장
  - 입고 이력 모든 건 기록

Sheet 4: 출고 대장
  - 출고 이력 모든 건 기록

Sheet 5: 대시보드
  - 발주 필요 품목 자동 표시 (현재고 < 발주점)
  - 재고 회전율 계산
  - 과재고 품목 표시
```

#### 발주 필요 자동 알림 수식
```excel
=IF(현재고!B2 <= 품목마스터!F2, "🔴 발주 필요",
   IF(현재고!B2 <= 품목마스터!E2 * 1.2, "⚠️ 주의", "✅ 정상"))
```

## 고급 패턴

### 바코드/QR 활용
```
바코드 시스템 구축 시:
1. 품목별 바코드/QR 생성 (무료: qr-code-generator.com)
2. 스마트폰/바코드 스캐너로 입출고 처리
3. 엑셀 VLOOKUP으로 자동 품목 정보 조회

저비용 구현:
- 구글 폼 + 바코드 스캐너 키보드 입력 방식
- 스마트폰 앱 활용 (Inventory Plus 등)
```

### 재고 손실 원인 분류
```
손실 유형:
① 분실/도난: 보안 강화, 책임 명확화
② 파손/불량: 취급 방법 개선, 포장 강화
③ 유통기한 초과: FEFO 관리, 선입선출 강화
④ 오기입: 이중 확인 프로세스

재고 손실률 = (손실량 / 총 입고량) × 100
목표: 0.5% 이하
```
