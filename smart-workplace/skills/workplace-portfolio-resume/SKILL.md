# 한국형 이력서 및 포트폴리오 작성 가이드

## 빠른 참조 (Quick Reference)

### 취업 서류 유형
| 서류 | 목적 | 분량 |
|------|------|------|
| 국문 이력서 | 기본 인적사항 및 경력 요약 | 1-2페이지 |
| 영문 이력서(CV) | 글로벌/외국계 기업 | 1-2페이지 |
| 자기소개서 | 지원 동기, 역량, 포부 | 4항목 기준 |
| 경력기술서 | 프로젝트/성과 상세 기술 | 2-5페이지 |
| 포트폴리오 | 실제 작업물/성과 증명 | PDF/온라인 |

### 자기소개서 4대 항목 (대기업 공통)
1. 성장 과정
2. 성격의 장단점
3. 지원 동기
4. 입사 후 포부

### 취업 서류 합격 핵심 원칙
- **수치화**: "개선했습니다" → "20% 향상시켰습니다"
- **구체성**: "마케팅 업무" → "SNS 콘텐츠 기획 및 월 팔로워 2,000명 증가"
- **직무 연관성**: 지원 직무와 관련 없는 경험은 최소화

## 이력서 작성 가이드

### 국문 이력서 표준 형식
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
이        력        서
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
성  명: {이름}                    사  진
생년월일: 1990년 X월 X일          (3.5 × 4.5cm)
연  락  처: 010-XXXX-XXXX
이  메  일: {이메일}
주  소: {주소}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▶ 학력 사항
기간             | 학교명               | 전공       | 학점/비고
----------------|---------------------|-----------|----------
2008.03-2012.02 | {대학교}             | {학과}      | X.XX/4.5
2005.03-2008.02 | {고등학교}           |            | 졸업

▶ 경력 사항
기간             | 회사명        | 부서/직책      | 담당 업무
----------------|--------------|--------------|----------
2019.01-현재    | {회사명}      | {팀}/{직책}    | {업무 요약}
2016.03-2018.12 | {회사명}      | {팀}/{직책}    | {업무 요약}

▶ 자격증/어학
- {자격증명}: {취득일}, {발급기관}
- {영어}: TOEIC {점수} ({취득년도})
- {기타 어학}: {내용}

▶ 수상/활동 (선택)
- {수상 내역}: {수상 기관}, {년도}
- {활동}: {내용}

▶ 스킬 (IT/직무 스킬)
- 프로그래밍: {언어/도구}
- 업무 도구: MS Office (고급), {기타}
- 기타: {특이 스킬}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 경력기술서 작성 (경력직 필수)
```
[경력 기술서]
이름: {이름}  지원 직무: {직무}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. {회사명} ({기간})
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
부서/직책: {팀}/{직책}
주요 담당 업무:
  - {업무 영역 1}: {내용}
  - {업무 영역 2}: {내용}

주요 프로젝트:

[프로젝트명 1] ({기간})
  - 역할: {역할}
  - 내용: {수행한 것}
  - 성과: {수치화된 결과}
    예) 기존 대비 처리 시간 40% 단축, 월 비용 1,500만 원 절감

[프로젝트명 2] ({기간})
  - 역할: {역할}
  - 내용: {수행한 것}
  - 성과: {결과}
```

## 자기소개서 작성 가이드

### STAR 기법 활용 (경험 서술)
```
S (Situation): 상황 설명
T (Task): 맡은 역할/과제
A (Action): 구체적 행동/방법
R (Result): 결과 및 배운 점

예시:
S: "대학 졸업 프로젝트에서 팀장을 맡아 5명의 팀을 이끌었습니다"
T: "6주 내에 앱 MVP를 개발해야 하는 상황에서 팀원 간 의견 충돌이 있었습니다"
A: "주 2회 정기 회의를 도입하고 각자의 역할을 명확히 재분배했습니다"
R: "프로젝트를 예정보다 1주일 빨리 완료했고, 교수 평가에서 A+를 받았습니다"
```

### 항목별 작성 템플릿

#### 성장 과정
```
저는 {지역/환경}에서 {부모님/가족}의 {가르침/영향으로} 자랐습니다.
특히 {구체적 경험이나 사건}을 통해 {핵심 가치/역량}을 키울 수 있었습니다.
이러한 배경이 저를 {현재의 자신의 특성}으로 성장하게 만들었습니다.
[200-300자]
```

#### 지원 동기 (가장 중요)
```
[기업 지원 동기 구조]
① 업계/회사에 대한 관심: "{회사/산업}이 {왜 중요한지}"
② 회사와 나의 연결점: "저의 {경험/역량}이 {회사 방향}과 일치합니다"
③ 구체적 기여 방안: "입사 후 {구체적으로 어떻게 기여할지}"

주의: "취업하고 싶어서", "복리후생이 좋아서" 절대 금지
```

#### 성격의 장단점
```
[장점 작성]
저의 강점은 {장점}입니다.
{구체적 경험에서 이 장점이 드러난 사례}
이를 통해 {성과/결과}를 달성했습니다.

[단점 작성]
제가 보완해야 할 점은 {단점}입니다.
이를 개선하기 위해 {구체적 노력}을 하고 있으며,
그 결과 {개선된 점/변화}를 경험했습니다.
```

## 구현 가이드

### 포트폴리오 구성 (직무별)
```
디자인/크리에이티브:
- PDF + 온라인 링크 (Behance, 개인 사이트)
- 최고 작업물 5-10개 선별
- 각 프로젝트: 목표 → 과정 → 결과 서술

개발/IT:
- GitHub 프로필 정비
- 주요 프로젝트 README 작성
- 기술 스택 명확히 표기

마케팅/기획:
- 수행 캠페인/프로젝트 결과물
- 성과 지표 (트래픽, 전환율, 매출)
- 담당 역할 명확히 기술
```

### ATS(채용 시스템) 최적화
```
국내 주요 채용 플랫폼:
- 사람인, 잡코리아: 자체 이력서 양식 사용
- 링크드인: 영문 위주, 글로벌 기업
- 원티드: 스타트업 친화, 간결한 형식
- 잡플래닛: 경력직 위주

키워드 최적화:
- JD(채용공고)의 핵심 키워드 이력서에 포함
- 직무 관련 스킬 명사로 명시 (Excel, Python, SQL 등)
- 동사는 성과 중심으로 (달성, 개선, 구축, 기획)
```

## 고급 패턴

### 경력직 핵심 차별화 포인트
```
신입: 잠재력, 배우려는 의지, 기초 역량
경력: 즉시 전력감, 구체적 성과, 문제 해결 능력

경력직 자소서 포인트:
- 숫자로 말하라: %, 금액, 건수, 시간 등
- 전 직장에서 이룬 성과 TOP 3 준비
- 이직 이유는 긍정적으로 표현
```

### 면접 준비 체크리스트
```
서류 합격 후:
□ 이력서/자소서 내용 숙지 (질문 대비)
□ 회사 홈페이지, 최근 뉴스 조사
□ 직무 관련 최신 트렌드 파악
□ 예상 질문 30개 + 답변 준비
□ 복장, 이동 시간 확인
```
