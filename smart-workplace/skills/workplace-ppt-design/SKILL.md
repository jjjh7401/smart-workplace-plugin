# 한국형 PPT 디자인 시스템

## 빠른 참조 (Quick Reference)

### 폰트 시스템
| 용도 | 폰트 | 크기 | 굵기 |
|------|------|------|------|
| 표지 제목 | 나눔명조 Bold / Pretendard | 44-54pt | Bold |
| 섹션 제목 | Pretendard SemiBold | 32-36pt | SemiBold |
| 슬라이드 제목 | Pretendard SemiBold | 24-28pt | SemiBold |
| 본문 | Pretendard Regular | 16-18pt | Regular |
| 캡션/주석 | Pretendard Light | 12-14pt | Light |

### 한국 기업 색상 팔레트
```
블루 계열 (신뢰/전문성):
  메인: #1B4F9E (진한 네이비)
  포인트: #2E7FD9 (밝은 파랑)
  배경: #EBF3FD (연한 파랑)

그레이 계열 (깔끔/세련):
  텍스트: #2C2C2C (진한 회색)
  서브: #6B7280 (중간 회색)
  배경: #F5F5F5 (밝은 회색)

포인트 컬러:
  강조: #E8481E (주황빨강, 경고/강조)
  성공: #16A34A (초록, 성과 강조)
  경고: #D97706 (주황, 주의 사항)
```

### 슬라이드 레이아웃 원칙
- 텍스트 최소화: 한 슬라이드 = 하나의 메시지
- 여백 충분히: 컨텐츠 면적 60%, 여백 40%
- 1-6-6 법칙: 1개 주제, 6줄 이하, 6단어 이하/줄

## 슬라이드 유형별 디자인

### 1. 표지 슬라이드 구성
```
배경: 진한 네이비(#1B4F9E) 또는 순백
[로고 위치: 좌상단 또는 우하단]

[중앙 배치]
━━━━━━━━━━━━━━━━
{메인 제목 - 나눔명조 Bold, 44pt, 흰색}
{부제목 - Pretendard, 20pt, 연한 색}
━━━━━━━━━━━━━━━━
{발표자 | 날짜 | 부서 - 14pt}
```

### 2. 목차 슬라이드
```
[좌측 1/3: 제목 영역]
CONTENTS
목차

[우측 2/3: 목차 항목]
01  {섹션 1} ────────────────── 05
02  {섹션 2} ────────────────── 12
03  {섹션 3} ────────────────── 18
04  {섹션 4} ────────────────── 25
```

### 3. 섹션 구분 슬라이드
```
배경: 메인 컬러(#1B4F9E) 풀스크린

[중앙]
01
{섹션 제목 - 흰색, 36pt, Bold}
Section Title
```

### 4. 텍스트 중심 슬라이드
```
[상단: 슬라이드 제목 - 28pt, 메인 컬러]
─────────────────────────────────────────

[본문 영역]
■ {핵심 포인트 1}
  ↳ {세부 설명 - 16pt, 그레이}

■ {핵심 포인트 2}
  ↳ {세부 설명}

■ {핵심 포인트 3}
  ↳ {세부 설명}

[하단: 페이지 번호 + 로고]
```

### 5. 데이터 시각화 슬라이드
```
[상단 제목]
{인사이트를 담은 제목: 예) "매출 전년비 23% 성장"}

[메인 차트 영역: 슬라이드 60% 차지]
[차트 종류 선택:
  - 시계열: 꺾은선
  - 비교: 막대
  - 구성: 파이/도넛
  - 상관: 산점도]

[하단 요약]
KEY TAKEAWAY: {차트의 핵심 메시지 한 줄}
```

### 6. 프로세스/흐름도 슬라이드
```
[제목]

[단계 표현: 화살표로 연결]
① {단계 1}  →  ② {단계 2}  →  ③ {단계 3}
[아이콘]     [아이콘]         [아이콘]
{설명}        {설명}           {설명}

또는 세로형:
STEP 1 ──── {내용}
    │
STEP 2 ──── {내용}
    │
STEP 3 ──── {내용}
```

### 7. 비교 슬라이드 (Before/After, AS-IS/TO-BE)
```
         AS-IS              TO-BE
    ┌─────────────┐    ┌─────────────┐
    │ 현재 상태   │ → │ 개선 후     │
    │ (연한 배경) │    │ (강조 배경) │
    │ • {특징 1} │    │ • {개선 1} │
    │ • {특징 2} │    │ • {개선 2} │
    └─────────────┘    └─────────────┘
```

## 구현 가이드

### PPT 제작 워크플로우
1. **스토리 구조 먼저**: 슬라이드 전 논리 흐름 설계
2. **텍스트 아웃라인**: 각 슬라이드의 핵심 메시지 1줄씩
3. **레이아웃 스케치**: 대략적인 배치 구상
4. **디자인 적용**: 색상/폰트/여백 통일
5. **검토**: 인쇄본 또는 다른 사람에게 확인

### 발표 유형별 슬라이드 수
| 발표 유형 | 시간 | 권장 슬라이드 수 |
|-----------|------|-----------------|
| 임원 보고 | 10-15분 | 10-15장 |
| 팀 내부 발표 | 20-30분 | 15-25장 |
| 고객사 제안 | 30-60분 | 20-40장 |
| 컨퍼런스 발표 | 20-40분 | 20-30장 |

## 고급 패턴

### 애니메이션 활용 원칙
- 화면 전환: Fade 또는 없음 (화려한 전환 지양)
- 텍스트 등장: 'Appear' 또는 'Fade' (순서 있는 내용 설명 시)
- 데이터 차트: 막대/선의 성장 애니메이션 효과적
- 원칙: 애니메이션이 내용을 방해하면 제거

### 인쇄 최적화
- 배경색 유지 또는 흰 배경 버전 별도 제작
- 폰트 임베딩 설정 확인
- PDF 변환 시 해상도 고품질 설정

### 온라인 발표 특화 팁
- 폰트 크기 최소 20pt (화면 공유 시 가독성)
- 진한 대비색 사용 (저해상도 화면 대비)
- 슬라이드 수 30% 감소 (집중력 저하 고려)
